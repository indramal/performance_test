# Bun SSR Test

Bun-based Server-Side Rendered React application for performance testing.

## Features
- Server-Side Rendering with React 19
- API endpoint at `/api/data`
- Static file serving
- Port: 3001

## Development

```bash
bun install
bun run dev
```

## Production

```bash
bun install
bun run start
```

## Build

```bash
bun run build
```
