# Deno Fresh SSR Test

Deno Fresh framework application for performance testing.

## Features
- Server-Side Rendering with Preact
- API route at `/api/data`
- Static file serving
- Port: 3003

## Development

```bash
deno task dev
```

## Production

```bash
deno task build
deno task start
```

## Requirements
- Deno 1.37+
